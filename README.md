# 🦖 恐龙生存游戏

一个基于HTML5的恐龙生存和战斗游戏。

## 🌐 在线玩

访问GitHub Pages: **https://zongtong999040.github.io/dinogame/**

## 🎮 游戏特性

### 🦕 恐龙系统
- 6种可选恐龙（霸王龙、三角龙、剑龙、甲龙、棘龙、迅猛龙）
- 独特的属性和攻击方式
- 成长系统：幼年 → 青年 → 亚成年 → 成年 → 老年

### 🗺️ 探索区域
- **原始森林**: 6种敌人（Level 1-5）
- **河流**: 4种水生敌人（Level 2-6），可恢复水分
- **竞技场**: 7种强大敌人（Level 5-10），需要成年阶段

### ⚔️ 战斗系统
- 17种不同的敌对生物
- 4种攻击方式（撕咬、撕扯、撕裂、尾巴攻击）
- 战斗中可切换攻击方式（按M键或点击按钮）
- 策略性战斗，根据敌人特点选择攻击方式

### 🎯 控制方式
- **WASD / 方向键**: 移动
- **空格键**: 攻击
- **M键**: 切换攻击方式
- **B键**: 打开/关闭菜单
- **鼠标点击**: 点击地面移动到目标位置

### 📈 成长系统
- 击败敌人获得经验值
- 收集肉类恢复生命值
- 在河流区域恢复水分
- 升级提升属性

## 🛠️ 技术栈

- 纯HTML5 + CSS3 + JavaScript
- Tailwind CSS（通过CDN）
- Font Awesome图标
- 响应式设计

## 🎯 游戏玩法

1. **选择恐龙**: 游戏开始时选择你喜欢的恐龙
2. **探索区域**: 在不同区域寻找敌人
3. **战斗**: 靠近敌人或按空格键开始战斗
4. **切换攻击**: 在战斗中按M键切换攻击方式
5. **收集资源**: 击败敌人获得肉类，恢复生命值
6. **成长**: 积累经验值，升级并解锁新区域
